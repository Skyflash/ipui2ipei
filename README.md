IPUI 2 IPEI
===========
###### by Gorka Hernandez

*IPUI 2 IPEI converts DECT phone IPUI codes to standard IPEI ones, all done in an ETSI compliant process.*
*Built using a modular design and latest Bootstrap version (v3.2.0)*
